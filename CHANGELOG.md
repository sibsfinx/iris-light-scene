# changelog

## latest

switched from 3d mesh to a particle system. dark petal body is now a low-opacity additive mesh, vein lines are tiny particles fanning out radially from the base of each petal, glare spots sit at the tips and edges to catch the bloom. motion reactiveness, slow rotation, flash/glare all still intact.

## before that

- added motion-reactive color — scene shifts hue when camera picks up movement
- crystal-glass iris with navy petals, iridescence, transmission, clearcoat
- god-ray lighting, deep DOF, bloom, grain, vignette
- mediapipe presence detection (motion / face / pose modes)
